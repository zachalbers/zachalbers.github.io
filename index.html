Campus MApp - Scheduler Extension

Summary

CampusMApp is an ongoing project that wants to make navigating campuses easier. I worked to develop for the app a way for users to upload their class schedule either by search selection or OCR. To accomplish this, my team and I first created a project plan after discussions with the CampusMApp staff to lay out realistic goals for our given timeline(1 semester). High-fidelity prototypes were then developed of what the scheduler could look like to recevie feedback from the clients as well as students(potential users). We then worked to develop the the scheduler in Xcode, which could then be implemented into the CampusMApp app.





My Roles in the Project

  • Discussed with CampusMApp managers and developers to determine the scope of the project and what our team should accomplish
  • Designed and developed high-fidelity prototypes of the scheduler in Adobe XD
  • Presented prototypes to students and other potential users in feedback sessions to get input on the designs, to improve the experience of using the schedule builder
  • In Swift, developed a large portion of the scheduler's structure, including the course search/selection views and controllors as well as parsing course data

Tools/Platforms Used

  • Xcode
  • Github

Prototypes Developed in Adobe XD

This project exposed my more to the agile development cycle as well as working with more specific requirements given by a client. Working with Adobe XD to create mockups of the scheduler expanded my design skills. I also improved my Swift developement skills and learned about integrating components from different developers into one app.

The Nanny Shack

Summary

A prototype website and mobile app built for an online nanny-booking service. The website provides a system where the schedule for nannies can be viewed and updated. Memberships are created for parents who can then see their financial information and their childrenâs schedule, and new users can check the availability of the daycare centre and then book a drop-in for their child. My team and my efforts focused first on the development of an Entity Relationship and Relational Model diagram for the system, that we then built our database management system from off of using PHP and MySQL. We also built a basic website and mobile app as well a user manual for our system.





My Roles in the Project

  • Designed the entity relationship model and helped implement the design using MySQL
  • Designed the mobile app using Xcode and Swift, including the networking logic for posting to the database
  • Developed a user manual for the website and system & tested the database with sample data records

Tools/Platforms Used

  • phpMyAdmin
  • Atom
  • Xcode
